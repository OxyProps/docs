This page lists all Shadows CSS custom properties available in the OxyProps Framework

## Shadow Parameters

```css
  --o-shadow-color
  --o-shadow-strength
  ```

## Shadows

```css
  --o-shadow-1
  --o-shadow-2
  --o-shadow-3
  --o-shadow-4
  --o-shadow-5
  --o-shadow-6
  ```

## Inner Shadows

```css
  --o-inner-shadow-0
  --o-inner-shadow-1
  --o-inner-shadow-2
  --o-inner-shadow-3
  --o-inner-shadow-4
```

## Font Weights as Shadows

```css
  --o-shadoweight-1
  --o-shadoweight-2
  --o-shadoweight-3
  --o-shadoweight-4
  --o-shadoweight-5
  --o-shadoweight-6
  --o-shadoweight-7
  --o-shadoweight-8
```
