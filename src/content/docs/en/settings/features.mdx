## Admin Preferences

These settings are used to customize the o-props Admin App.

### Debounce Time

The debounce time is the time in milliseconds that o-props will wait before saving your changes after you stop typing.

If you feel like o-props is saving too often and typing is incomfortable, you can increase this value.

## In the editor

### Context Menus

o-props provides context menus in the editor to quickly populate CSS value fields with the framework props.

You can decide to disable these context menus if you don't use them or if you prefer to use you keyboard.
