This page lists all Border Sizes CSS custom properties and utility classes available in the OxyProps Framework

## Border sizes

| Description              | CSS Variable                                                                      | Utility Class                                                  |
| ------------------------ | --------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| Creates a thin border.   | <ClipboardCopy value="var(--o-border-size-1)">`--o-border-size-1`</ClipboardCopy> | <ClipboardCopy value="o-border-1">`o-border-1`</ClipboardCopy> |
| Creates a light border.  | <ClipboardCopy value="var(--o-border-size-2)">`--o-border-size-2`</ClipboardCopy> | <ClipboardCopy value="o-border-2">`o-border-2`</ClipboardCopy> |
| Creates a medium border. | <ClipboardCopy value="var(--o-border-size-3)">`--o-border-size-3`</ClipboardCopy> | <ClipboardCopy value="o-border-3">`o-border-3`</ClipboardCopy> |
| Creates a thick border.  | <ClipboardCopy value="var(--o-border-size-4)">`--o-border-size-4`</ClipboardCopy> | <ClipboardCopy value="o-border-4">`o-border-4`</ClipboardCopy> |
| Creates a heavy border.  | <ClipboardCopy value="var(--o-border-size-5)">`--o-border-size-5`</ClipboardCopy> | <ClipboardCopy value="o-border-5">`o-border-5`</ClipboardCopy> |
