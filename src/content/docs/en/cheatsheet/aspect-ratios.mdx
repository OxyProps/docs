This page lists all Aspect Ratio CSS custom properties and utility classes available in the OxyProps Framework

## Aspect Ratios

| Description                                | CSS Variable                                                                            | Utility Class                                                      |
| ------------------------------------------ | --------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |

| Applies a portrait aspect ratio of 3:4.    | <ClipboardCopy value="var(--o-ratio-portrait)">`--o-ratio-portrait`</ClipboardCopy>     | <ClipboardCopy value="o-portrait">`o-portrait`</ClipboardCopy>     |
| Applies a widescreen aspect ratio of 16:9. | <ClipboardCopy value="var(--o-ratio-widescreen)">`--o-ratio-widescreen`</ClipboardCopy> | <ClipboardCopy value="o-widescreen">`o-widescreen`</ClipboardCopy> |
| Applies a ultrawide aspect ratio of 18:5.  | <ClipboardCopy value="var(--o-ratio-ultrawide)">`--o-ratio-ultrawide`</ClipboardCopy>   | <ClipboardCopy value="o-ultrawide">`o-ultrawide`</ClipboardCopy>   |
| Applies a golden aspect ratio of 1.618:1.  | <ClipboardCopy value="var(--o-ratio-golden)">`--o-ratio-golden`</ClipboardCopy>         | <ClipboardCopy value="o-golden">`o-golden`</ClipboardCopy>         |
