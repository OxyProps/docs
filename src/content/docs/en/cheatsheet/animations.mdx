This page lists all Animations CSS custom properties and utility classes available in the OxyProps Framework

## Target CSS properties

The animation CSS custom properties include shorthand values for:
- `animation-name`
- `animation-duration`
- `animation-timing-function`

Therefore, they must be used on the `animation` shorthand CSS property.

- [`animation`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation)

Example custom CSS:

<Tabs>
    <TabItem label="bricks">
```css
%root% {
    animation: var(--o-animation-fade-in) forwards;
}
```
    </TabItem>
    <TabItem label="oxygen">
```css title=".loaded Custom CSS"
animation: var(--o-animation-fade-in) forwards;
```
    </TabItem>
</Tabs>
