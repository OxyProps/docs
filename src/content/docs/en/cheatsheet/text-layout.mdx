This page lists all Text Layout CSS custom properties available in the OxyProps Framework

## Text Orientation

| Description                        | Utility Class                                                    |
| ---------------------------------- | ---------------------------------------------------------------- |
| Vertical top to bottom             | <ClipboardCopy value="o-text-down">`o-text-down`</ClipboardCopy> |
| Vertical bottom to top             | <ClipboardCopy value="o-text-up">`o-text-up`</ClipboardCopy>     |
| With upright character orientation | <ClipboardCopy value="o-upright">`o-upright`</ClipboardCopy>     |

## Text Direction

| Description   | Utility Class                                        |
| ------------- | ---------------------------------------------------- |
| Left to Right | <ClipboardCopy value="o-ltr">`o-ltr`</ClipboardCopy> |
| Right to Left | <ClipboardCopy value="o-rtl">`o-rtl`</ClipboardCopy> |

## Text Alignment

| Description | Utility Class                                                          |
| ----------- | ---------------------------------------------------------------------- |
| Left        | <ClipboardCopy value="o-text-left">`o-text-left`</ClipboardCopy>       |
| Right       | <ClipboardCopy value="o-text-right">`o-text-right`</ClipboardCopy>     |
| Center      | <ClipboardCopy value="o-text-center">`o-text-center`</ClipboardCopy>   |
| Justify     | <ClipboardCopy value="o-text-justify">`o-text-justify`</ClipboardCopy> |
| Start       | <ClipboardCopy value="o-text-start">`o-text-start`</ClipboardCopy>     |
| End         | <ClipboardCopy value="o-text-end">`o-text-end`</ClipboardCopy>         |
