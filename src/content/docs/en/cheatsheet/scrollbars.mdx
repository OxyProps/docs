This page lists all Scrollbars CSS custom properties and utility classes available in the OxyProps Framework

## Scrollbars

| Description | Utility Class                                                    |
| ----------- | ---------------------------------------------------------------- |
| Base class  | <ClipboardCopy value="o-scrollbar">`o-scrollbar`</ClipboardCopy> |

## Scrollbars Colors

| Description   | Utility Class                                                                    |
| ------------- | -------------------------------------------------------------------------------- |
| Brand color   | <ClipboardCopy value="o-scrollbar-brand">`o-scrollbar-brand`</ClipboardCopy>     |
| Accent color  | <ClipboardCopy value="o-scrollbar-accent">`o-scrollbar-accent`</ClipboardCopy>   |
| Links color   | <ClipboardCopy value="o-scrollbar-links">`o-scrollbar-links`</ClipboardCopy>     |
| Visited color | <ClipboardCopy value="o-scrollbar-visited">`o-scrollbar-visited`</ClipboardCopy> |

## Scrollbars Shape

| Description | Utility Class                                                                      |
| ----------- | ---------------------------------------------------------------------------------- |
| No Radius   | <ClipboardCopy value="o-scrollbar-noradius">`o-scrollbar-noradius`</ClipboardCopy> |
| Thin        | <ClipboardCopy value="o-scrollbar-thin">`o-scrollbar-thin`</ClipboardCopy>         |
| Thick       | <ClipboardCopy value="o-scrollbar-thick">`o-scrollbar-thick`</ClipboardCopy>       |

## Customize the Scrollbars

You can customize the Scrollbars by using the following CSS custom properties:

| CSS Custom Property                                                                                | Default Value                   |
| -------------------------------------------------------------------------------------------------- | ------------------------------- |
| <ClipboardCopy value="--o_scrollbar-width">`--o_scrollbar-width`</ClipboardCopy>                   | calc(var(--o-size-fluid-1) / 2) |
| <ClipboardCopy value="--o_scrollbar-height">`--o_scrollbar-height`</ClipboardCopy>                 | calc(var(--o-size-fluid-1) / 2) |
| <ClipboardCopy value="--o_scrollbar-thumb-radius">`--o_scrollbar-thumb-radius`</ClipboardCopy>     | var(--o-radius-round)           |
| <ClipboardCopy value="--o_scrollbar-thumb-bg">`--o_scrollbar-thumb-bg`</ClipboardCopy>             | var(--o-scrollbar-color)        |
| <ClipboardCopy value="--o_scrollbar-track-radius ">`--o_scrollbar-track-radius `</ClipboardCopy  > | 0                               |
| <ClipboardCopy value="--o_scrollbar-track-bg">`--o_scrollbar-track-bg`</ClipboardCopy    >         | transparent                     |
