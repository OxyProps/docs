## Fixed Gaps

:::tip[Negative Gaps?]
Negative values are invalid for `gap`, `row-gap`, and `column-gap`. This is why there are no `o-gap-fixed-000` and `o-gap-fixed-00` classes. If you need to use negative values, use margin classes on the children elements.
:::
