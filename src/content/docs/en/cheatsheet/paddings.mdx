## Fluid Paddings (Top, Right, Bottom, Left)

:::tip[Wording]
I use the terms Top, Right, Bottom, Left to describe the padding directions. This is for being easily understood by everyone. However, the actual CSS properties are `padding-block-start`, `padding-inline-end`, `padding-block-end`, `padding-inline-start`. The `start` and `end` are relative to the writing mode. For example, if the writing mode is `ltr`, the `start` is `left` and the `end` is `right`. If the writing mode is `rtl`, the `start` is `right` and the `end` is `left`.
