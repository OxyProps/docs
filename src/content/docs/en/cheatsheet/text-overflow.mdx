This page lists all Text Overflow CSS custom properties available in the OxyProps Framework

## Text Overflow

| Description                                                   | Utility Class                                                                              |
| ------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| Excess text is clipped and not displayed                      | <ClipboardCopy value="o-overflow-text-clip">`o-overflow-text-clip`</ClipboardCopy>         |
| Excess text is truncated and an ellipsis ("...") is displayed | <ClipboardCopy value="o-overflow-text-ellipsis">`o-overflow-text-ellipsis`</ClipboardCopy> |
| Limit displayed text to 1 line.                               | <ClipboardCopy value="o-line-clamp-1">`o-line-clamp-1`</ClipboardCopy>                     |
| Limit displayed text to 2 lines.                              | <ClipboardCopy value="o-line-clamp-2">`o-line-clamp-2`</ClipboardCopy>                     |
| Limit displayed text to 3 lines.                              | <ClipboardCopy value="o-line-clamp-3">`o-line-clamp-3`</ClipboardCopy>                     |
| Limit displayed text to 4 lines.                              | <ClipboardCopy value="o-line-clamp-4">`o-line-clamp-4`</ClipboardCopy>                     |
| Limit displayed text to 5 lines.                              | <ClipboardCopy value="o-line-clamp-5">`o-line-clamp-5`</ClipboardCopy>                     |
| Limit displayed text to 6 lines.                              | <ClipboardCopy value="o-line-clamp-6">`o-line-clamp-6`</ClipboardCopy>                     |
| Limit displayed text to 7 lines.                              | <ClipboardCopy value="o-line-clamp-7">`o-line-clamp-7`</ClipboardCopy>                     |
| Limit displayed text to 8 lines.                              | <ClipboardCopy value="o-line-clamp-8">`o-line-clamp-8`</ClipboardCopy>                     |
| Limit displayed text to 9 lines.                              | <ClipboardCopy value="o-line-clamp-9">`o-line-clamp-9`</ClipboardCopy>                     |

## Word Breaks

| Description                                                     | Utility Class                                                                          |
| --------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| The default behavior is used.                                   | <ClipboardCopy value="o-word-wrap-normal">`o-word-wrap-normal`</ClipboardCopy>         |
| Words are broken and wrapped to fit within the available space. | <ClipboardCopy value="o-word-wrap-anywhere">`o-word-wrap-anywhere`</ClipboardCopy>     |
| Allows words to break between any two letters.                  | <ClipboardCopy value="o-word-wrap-break-word">`o-word-wrap-break-word`</ClipboardCopy> |
| No special rules for line-breaking should be applied.           | <ClipboardCopy value="o-word-break-normal">`o-word-break-normal`</ClipboardCopy>       |
| Allows words to break anywhere in a line.                       | <ClipboardCopy value="o-word-break-break-all">`o-word-break-break-all`</ClipboardCopy> |
| Prevents breaks between words, but allows breaks within words.  | <ClipboardCopy value="o-word-break-keep-all">`o-word-break-keep-all`</ClipboardCopy>   |
