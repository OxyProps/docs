:::note[Patterns]
CSS Variable: `--o-radius-{1-6 | round}`  
Utility Class: `o-radius-{ side | corner }-{ 1-6 | round }`
:::

```css title="OxyProps CSS Variables"
  --o-radius-1: 2px;
	--o-radius-2: 5px;
	--o-radius-3: 1rem;
	--o-radius-4: 2rem;
	--o-radius-5: 4rem;
	--o-radius-6: 8rem;
	--o-radius-round: 1e5px;
```

## No Radius

<div style="aspect-ratio: 1; height: 10rem; border: 1px solid var(--sl-color-accent-high); display:grid; place-items:center;">o-noradius</div>

Use this class to remove the border radius from an element.

```html title="Utility Class"
  o-noradius
```
