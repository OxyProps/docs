The button has a default border color of `var(--o-surface-3)` in dark mode, `var(--o-surface-1)` in light mode. You can change this value by using the following CSS custom properties:

| description                         | custom property                                                                         |
| ----------------------------------- | --------------------------------------------------------------------------------------- |
| Border color in light mode          | <ClipboardCopy value="--o_border-light:">`--o_border-light`</ClipboardCopy>             |
| Border color in light mode on hover | <ClipboardCopy value="--o_border-light-hover:">`--o_border-light-hover`</ClipboardCopy> |
| Border color in dark mode           | <ClipboardCopy value="--o_border-dark:">`--o_border-dark`</ClipboardCopy>               |
| Border color in dark mode on hover  | <ClipboardCopy value="--o_border-dark-hover:">`--o_border-dark-hover`</ClipboardCopy>   |
| Border color                        | <ClipboardCopy value="--o_border:">`--o_border`</ClipboardCopy>                         |
| Border color on hover               | <ClipboardCopy value="--o_border-hover:">`--o_border-hover`</ClipboardCopy>             |

```css title="Fancy button site wide"
.o-btn {
  --o_border-light: var(--o-accent);
  --o_border-dark: var(--o-visited);
  --o_border-light-hover: var(--o-visited);
  --o_border-dark-hover: var(--o-accent);
}
```

```css title="Fancy button modifier class"
.o-btn-fancy {
  --o_border-light: var(--o-accent);
  --o_border-dark: var(--o-visited);
  --o_border-light-hover: var(--o-visited);
  --o_border-dark-hover: var(--o-accent);
}
```
