The button has a default padding inline of `1.75ch` and a default padding block of `0.75ch`. You can change this value by using the `--o_padding-inline` and `--o_padding-block` custom properties.

| description    | custom property                                                                 |
| -------------- | ------------------------------------------------------------------------------- |
| Padding inline | <ClipboardCopy value="--o_padding-inline:">`--o_padding-inline`</ClipboardCopy> |
| Padding block  | <ClipboardCopy value="--o_padding-block:">`--o_padding-block`</ClipboardCopy>   |

```css title="Narrow button site wide"
.o-btn {
  --o_padding-inline: 1ch;
  --o_padding-block: 0.5ch;
}
```

```css title="Fat button modifier class"
.o-btn-fat {
  --o_padding-inline: 5ch;
  --o_padding-block: 2ch;
}
```
