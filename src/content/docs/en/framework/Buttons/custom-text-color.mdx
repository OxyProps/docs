The button has a default text color of `var(--o-text-1)` in dark mode, `var(--o-text-1)` in light mode. You can change this value by using the following CSS custom properties:
| description | custom property |
| --- | --- |
| Text color in light mode |<ClipboardCopy value="--o_text-light:">`--o_text-light`</ClipboardCopy> |
| Text color in light mode on hover |<ClipboardCopy value="--o_text-light-hover:">`--o_text-light-hover`</ClipboardCopy> |
| Text color in dark mode |<ClipboardCopy value="--o_text-dark-hover:">`--o_text-dark-hover`</ClipboardCopy> |
| Text color in dark mode on hover |<ClipboardCopy value="--o_text-dark-hover:">`--o_text-dark-hover`</ClipboardCopy> |
| Text color |<ClipboardCopy value="--o_text:">`--o_text`</ClipboardCopy> |

    ```css title="Custom text color site wide"
    .o-btn {
      --o_text-light: forestgreen;
      --o_text-light-hover: darkgreen;
      --o_text-dark: chocolate;
      --o_text-dark-hover: coral;
    }
    ```
    ```css title="Custom text color modifier class"
    .o-btn-fancy {
      --o_text-light: forestgreen;
      --o_text-light-hover: darkgreen;
      --o_text-dark: chocolate;
      --o_text-dark-hover: coral;
    }
    ```
