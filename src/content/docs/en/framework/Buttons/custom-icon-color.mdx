The button has a default icon color of `var(--o-accent-700)` in dark mode, `var(--o-accent-050)` in light mode. This is the color used to style the icon if you use one. You can change this value by using the following CSS custom properties:
| description | custom property |
| --- | --- |
| Icon color in light mode |<ClipboardCopy value="--o_icon-color-light:">`--o_icon-color-light`</ClipboardCopy> |
| Icon color in light mode on hover |<ClipboardCopy value="--o_icon-color-light-hover:">`--o_icon-color-light-hover`</ClipboardCopy> |
| Icon color in dark mode |<ClipboardCopy value="--o_icon-color-dark-hover:">`--o_icon-color-dark-hover`</ClipboardCopy> |
| Icon color in dark mode on hover |<ClipboardCopy value="--o_icon-color-dark-hover:">`--o_icon-color-dark-hover`</ClipboardCopy> |
| Icon color |<ClipboardCopy value="--o_icon-color:">`--o_icon-color`</ClipboardCopy> |
