The button has a default border thickness of `--o-border-size-2`. You can change this value by using the `--o_border-size` custom property.

<div style="display: flex; flex-wrap: wrap; column-gap:5rem;row-gap:1rem; width:100%; justify-content: center; align-items: center;padding-block: 4rem;">
    <a href="#custom-border-size" class="o-btn o-btn-outline o-btn-links" style="--o_border-size: 0;"><span>button</span></a>
    <a href="#custom-border-size" class="o-btn o-btn-outline o-btn-links" style="--o_border-size: var(--o-border-size-1);"><span>button</span></a>
    <a href="#custom-border-size" class="o-btn o-btn-outline o-btn-links"><span>button</span></a>
    <a href="#custom-border-size" class="o-btn o-btn-outline o-btn-links" style="--o_border-size: var(--o-border-size-3);"><span>button</span></a>
</div>
    
    ```css title="Borderless button site wide"
    .o-btn {
      --o_border-size: 0;
    }
    ```
    ```css title="Thick border modifier class"
    .o-btn-thick-border {
      --o_border-size: var(--o-border-size-4);
    }
    ```
