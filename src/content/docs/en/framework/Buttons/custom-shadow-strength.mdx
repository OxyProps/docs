The button has a default shadow strength of `--o-shadow-strength-light` in light mode and `--o-shadow-strength-dark` as they are defined in you plugin dashboard. You can change this value by using the following CSS custom property:

The expected value is a percentage between `0%` and `100%`.

| description                   | custom property                                                                               |
| ----------------------------- | --------------------------------------------------------------------------------------------- |
| Shadow strength in light mode | <ClipboardCopy value="--o_shadow-strength-light:">`--o_shadow-strength-light`</ClipboardCopy> |
| Shadow strength in dark mode  | <ClipboardCopy value="--o_shadow-strength-dark:">`--o_shadow-strength-dark`</ClipboardCopy>   |
| Shadow strength               | <ClipboardCopy value="--o_shadow-strength:">`--o_shadow-strength`</ClipboardCopy>             |

    ```css title="Custom shadow strength site wide"
    .o-btn {
      --o_shadow-strength-light: 50%;
      --o_shadow-strength-dark: 50%;
    }
    ```
    ```css title="Custom shadow strength modifier class"
    .o-btn-fancy {
      --o_shadow-strength-light: 100%;
      --o_shadow-strength-dark: 100%;
    }
    ```
:::tip
You typically want to set a relatively low value for the shadow strength in light mode, something less than 10%. While in dark mode you can use a higher value, something between 50% and 90%.
:::
