The button has a default icon size of `2ch`. You can change this value by using the following CSS custom property:

| description | custom property                                                       |
| ----------- | --------------------------------------------------------------------- |
| Icon size   | <ClipboardCopy value="--o_icon-size:">`--o_icon-size`</ClipboardCopy> |

    ```css title="Custom icon size site wide"   
    .o-btn {
      --o_icon-size: 3ch;
    }
    ```
    ```css title="Custom icon size modifier class"
    .o-btn-fancy {
      --o_icon-size: 3ch;
    }
    ```
:::tip
You can use any unit of measurement for the icon size. The default is `ch` which is the height of the letter `0` in the current font. This is a good default because it scales with the font size.
:::
