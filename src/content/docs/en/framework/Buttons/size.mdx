## Size
The button is built in a way that it scales automatically with the text size. This means that if you change the font size of the button, the button will scale accordingly.

## Padding Customization

If you want to customize the padding of the button, you can use the custom padding CSS variable. Check the [dedicated padding customization page](/framework/buttons/custom-padding).
