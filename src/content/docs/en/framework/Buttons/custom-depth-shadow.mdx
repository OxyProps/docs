The button has a default depth shadow of `0 1px var(--o_border-light)` in light mode and `0 1px var(--o_border-dark)` in dark mode. This shadow adds a subtle line under the button giving a slight 3D effect. You can change this value by using the following CSS custom property:

The expected value is a CSS box-shadow value.

| description                | custom property                                                                         |
| -------------------------- | --------------------------------------------------------------------------------------- |
| Depth shadow in light mode | <ClipboardCopy value="--o_shadow-depth-light:">`--o_shadow-depth-light`</ClipboardCopy> |
| Depth shadow in dark mode  | <ClipboardCopy value="--o_shadow-depth-dark:">`--o_shadow-depth-dark`</ClipboardCopy>   |
| Depth shadow               | <ClipboardCopy value="--o_shadow-depth:">`--o_shadow-depth`</ClipboardCopy>             |

    ```css title="Custom depth shadow site wide"
    .o-btn {
      --o_shadow-depth-light: 0 1px var(--o_border-light);
      --o_shadow-depth-dark: 0 1px var(--o_border-dark);
    }
    ```
    ```css title="Custom depth shadow modifier class"
    .o-btn-fancy {
      --o_shadow-depth-light: 0 1px var(--o_border-light);
      --o_shadow-depth-dark: 0 1px var(--o_border-dark);
    }
    ```
