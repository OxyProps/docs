:::note[Patterns]
CSS Variable: `--o-border-size-{ 1-5 }`  
Utility Class: `o-border-{ "" | top | bottom | left | right }-{ 1-5 }`
:::

```css title="OxyProps CSS Variables"
  --o-border-size-1: 1px;
	--o-border-size-2: 2px;
	--o-border-size-3: 5px;
	--o-border-size-4: 10px;
	--o-border-size-5: 25px;
```
