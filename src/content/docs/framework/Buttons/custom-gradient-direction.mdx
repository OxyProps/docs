The button has a default gradient direction of `to bottom` in dark mode, `to top` in light mode. You can change this value by using the following CSS custom properties:

| description                      | custom property                                                                                               |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Gradient direction in light mode | <ClipboardCopy value="--o_gradient-direction-light:">`--o_gradient-direction-light`</ClipboardCopy>           |
| Gradient direction in dark mode  | <ClipboardCopy value="--o_gradient-direction-dark:">`--o_gradient-direction-dark`</ClipboardCopy> |
| Gradient direction               | <ClipboardCopy value="--o_gradient-direction:">`--o_gradient-direction`</ClipboardCopy>                       |

    <a href="#custom-gradient-direction" class="o-btn o-btn-brand-gr" style="--o_ink-shadow-light: none; --o_gradient-direction-light: to left;--o_gradient-direction-dark: to right;"><span>button</span></a>
    <a href="#custom-gradient-direction" class="o-btn o-btn-brand-gr" style="--o_ink-shadow-light: none; --o_gradient-direction-light: to bottom left;--o_gradient-direction-dark: to top right;"><span>button</span></a>
</div>
    
    ```css title="Custom gradient direction site wide"  
    .o-btn {
      --o_gradient-direction-light: to right;
      --o_gradient-direction-dark: to left;
    }
    ```
    ```css title="Custom gradient direction modifier class"
    .o-btn-fancy {
      --o_gradient-direction-light: to right;
      --o_gradient-direction-dark: to left;
    }
    ```
