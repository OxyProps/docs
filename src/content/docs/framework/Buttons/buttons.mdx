Buttons are an essential element of any website or application. They allow users to perform actions or navigate through your interface. This series of CSS properties and utility classes is designed to be flexible and easy to use.

## TL, DR

To style an element as a button, use the <ClipboardCopy value="o-btn">`o-btn`</ClipboardCopy> class.

```html title="Give base styling to a button"
<a href="#" class="o-btn">o-btn</a>
```

To add specific styles to a button, add a modifier class to the <ClipboardCopy value="o-btn">`o-btn`</ClipboardCopy> class.

```html title="Create a Success button"
<a href="#" class="o-btn o-btn-success">o-btn-success</a>
```

```html title="Create an Outline button"
<a href="#" class="o-btn o-btn-outline">o-btn-success</a>
```

To customize the buttons, use the buttons CSS custom properties.

If you modify the <ClipboardCopy value="o-btn">`o-btn`</ClipboardCopy> class, you will affect all buttons in your site.

```css title="Customize the buttons"
/* All buttons will have square corners and a highlight size of 1rem */
.o-btn {
	--o_border-radius: 0px;
	--o_highlight-size-hover: var(--o-size-3);
}
```

You can also create your own modifier classes to add specific styles to a button.

```css title="Create a custom button"
.o-btn-custom {
	--o_bg-light: var(--o-color-brand-100);
	--o_bg-dark: var(--o-color-brand-100);
	--o_text-light: var(--o-color-brand-900);
	--o_text-dark: var(--o-color-brand-900);
	--o_border-radius: 0px;
	--o_highlight-size-hover: var(--o-size-3);
}
```

```html title="Use the custom button"
<a href="#" class="o-btn o-btn-custom">o-btn-custom</a>
```

## Buttons or links ?

A button is a user interface element that is used to trigger an action, such as submitting a form or navigating to a new page. It is typically represented as a rectangular or rounded rectangle shape that can be clicked or tapped. Buttons can be created using HTML and CSS or through programming languages such as JavaScript.

On the other hand, a link styled as a button is a hyperlink that is designed to look like a button. It can be created by applying CSS styles to a standard hyperlink, such as changing the background color, border, and font size. When clicked, a link styled as a button can also trigger an action, such as opening a new page or submitting a form.

The main difference between a button and a link styled as a button is their intended use. Buttons are best used for actions that change the state of an application, such as submitting a form, saving data, or deleting content. Links styled as buttons, on the other hand, are best used for actions that navigate to a different page or location, such as going to a new section of a website or opening a new window.

In general, if the user is performing an action that changes the state of an application, a button is the appropriate choice. If the user is navigating to a different page or location, a link styled as a button may be a better choice.

### CSS icon

#### Step 1

Browse the icon library of your choice and copy the URL of the icon you want to use. In this example, we will use the [Iconify](https://icon-sets.iconify.design/) library.

<Image
      src={iconify}
      alt="Getting an icon from the Iconify Library."
      format="avif"
      quality="mid"
      width="900"
      height="450"
    />

Once you have found your icon:

1. Click the SVG button.
2. Click the SVG as data: URI button.
3. Click the copy button to copy the code to your clipboard.

#### Step 2

Create a custom CSS rule in your global CSS using the following model. This way, you insert the SVG code once and can reuse it without duplication.

```css
[data-icon='icon-name'] {
	--o_data-icon: url('data:image/svg+xml,%3Csvg xmlns="http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg" width="24" height="24" viewBox="0 0 24 24"%3E%3Cpath fill="currentColor" d="M11 17h2v-6h-2v6Zm1-8q.425 0 .713-.288T13 8q0-.425-.288-.713T12 7q-.425 0-.713.288T11 8q0 .425.288.713T12 9Zm0 13q-3.475-.875-5.738-3.988T4 11.1V5l8-3l8 3v6.1q0 3.8-2.263 6.913T12 22Z"%2F%3E%3C%2Fsvg%3E')
		center / contain no-repeat;
}
```

#### Step 3
