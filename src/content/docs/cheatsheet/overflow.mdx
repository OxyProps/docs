| Hidden      | <ClipboardCopy value="o-overflow-hidden">`o-overflow-hidden`</ClipboardCopy>   |
| Clip        | <ClipboardCopy value="o-overflow-clip">`o-overflow-clip`</ClipboardCopy>       |
| Scroll      | <ClipboardCopy value="o-overflow-scroll">`o-overflow-scroll`</ClipboardCopy>   |

| Overlay     | <ClipboardCopy value="o-overflow-overlay">`o-overflow-overlay`</ClipboardCopy> |
