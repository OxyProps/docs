This page lists all Accessibility CSS custom properties and utility classes available in the OxyProps Framework

## Screen Readers

| Description                              | Utility Class                                                        |
| ---------------------------------------- | -------------------------------------------------------------------- |
| Screen Reader Only                       | <ClipboardCopy value="o-sr-only">`o-sr-only`</ClipboardCopy>         |
| Not Screen Reader Only (Cancels sr-only) | <ClipboardCopy value="o-not-sr-only">`o-not-sr-only`</ClipboardCopy> |
