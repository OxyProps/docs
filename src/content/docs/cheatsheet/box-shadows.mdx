This page lists all Box Shadows CSS custom properties available in the OxyProps Framework

## Shadow Parameters

| Description                                                    | CSS Variable                                                                          |
| -------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| Defines the base color of the shadow. Use any valid CSS color. | <ClipboardCopy value="var(--o-shadow-color)">`--o-shadow-color`</ClipboardCopy>       |
| Defines the strength of the shadow as a percentage.            | <ClipboardCopy value="var(--o-shadow-strength)">`--o-shadow-strength`</ClipboardCopy> |
